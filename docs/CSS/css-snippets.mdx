### Center hoàn toàn

```scss
.item {
  position: relative;
  // Ở đây thì mới chỉ là topleft corner của item nằm ở center của container
  top: 50%;
  left: 50%;
  // -> cần transform theo item size để hoàn toàn center item theo container
  transform: translate(-50%, -50%);
}
```

Cùi hơn thì dùng flexbox hoặc `mx-auto` (chỉ dùng dc cho ~ element có declared width)

### Image scale with Container width

```css
.img-fluid {
  width: 100%;
  height: auto;
}
```

### Grid 2 columns

```css
.grid {
  /* min-width = 0, max-width = 1fr, Similar to Tailwind */
  grid-template-columns: repeat(2, minmax(0, 1fr))

  /* ❌ Wrong, == minmax(auto, 1fr), so the grid item is free to resize to 0 */
  grid-template-columns: 1fr
}
```

### Width: `min-content` vs `max-content` vs `fit-content`

```css
.box {
  width: fit-content;

  /* ... is the same as ... */
  width: auto;
  min-width: min-content;
  max-width: max-content;
}
```

### Aspect ratio

```css
.product-image {
  aspect-ratio: 1/1;
}
/* Or */
.product-image {
  /* 1:1 Aspect Ratio, e.g 16:9 will have the padding-top at 56.25%  */
  padding-top: 100%;
  position: relative;
}
.product-image > img {
  position: absolute;
  top: 0;
  left: 0;
  bottom: 0;
  right: 0;
  max-height: 100%;
  margin: auto;
}
```

### Hidden & Visible element

```css
div {
  display: none;
  transition: opacity 1s ease-out;
  opacity: 0;
}
div.active {
  opacity: 1;
  display: block;
}
```

### Margin best practice for Component

```jsx
// Hardcode margin in component
const Bad = ({ children }) => <div className="mb-2">{children}</div>

// Have a `className` prop so that we can dynamic change the margin
const Good = ({ children, className }) => (
  <div className={className}>{children}</div>
)
```

### `:not(:last-of-type)` for list item

```css
li:not(:last-of-type) {
  margin-bottom: 1rem;
}
```

### Hover cho `after/before`: pseudo class đứng trước pseudo element
